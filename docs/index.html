Copy Title & URL in Markdown format

About

This program is an extension for Chrome. Copy the page title and URL in Markdown format from the browser right-click.

Usage

  1. Right click in chrome page.
  2. Select Copy [Title](URL) > Copy [Title](URL), Copy Ttile
  3. Copied!
